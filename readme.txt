About: 

This repository holds several components for a baseline Azure Application with DevSecOps CI/CD Pipeline.   

The base open source project can be found at: https://github.com/DPD-30/devStack 

See readme.txt in each component for details on use.   
All components except bootstrap are built and deployed via Jenkins and the jenkinsfile in those components
shows the code used to do the builds.
